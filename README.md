# Cuzzie-Movers

Created for ITSD6.348 Systems Analysis at EIT Tairawhiti

This documentation will aim to analyse the content of an information system developed and used by Cuzzie Movers, a small New Zealand moving company. We will be checking for any issues associated with the system as a whole, as well as its sub-systems. The main methodology we will be using is the Software Development Life Cycle (SDLC), to allow for errors to be easily caught early in the project. This information system is split into three sub-systems: Human Resources, Customer Relations & Inventory. Work surrounding the Human Resources sub-system will be done by Shawn, the Customer Relations work will be completed by Ben, and the Inventory work will be completed by both of us.
